# <a name="bookmarkenum"></a>BookmarkEnum
指定指示操作开始位置的书签。  
  
|返回的常量|Value|说明|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|从当前记录开始。|  
|**adBookmarkFirst**|1|从第一条记录开始。|  
|**adBookmarkLast**|2|从最后一条记录开始。|
