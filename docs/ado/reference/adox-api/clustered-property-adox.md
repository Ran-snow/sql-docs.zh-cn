# <a name="clustered-property-adox"></a>Clustered 属性 (ADOX)
指示索引是否为聚集索引。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回一个 **Boolean** 值。  
  
## <a name="remarks"></a>备注  
 默认值为 **False**。  
  
 对于已追加到集合的 [索引](./index-object-adox.md) 对象，此属性是只读的。  
  
## <a name="applies-to"></a>应用于  
 [索引对象 (ADOX)](./index-object-adox.md)
