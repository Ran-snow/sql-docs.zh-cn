# <a name="type-property-key-adox"></a>Type 属性（项）(ADOX)
指示密钥的类型。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回可以是 [KeyTypeEnum](./keytypeenum.md)常量之一的 **Long** 值。 默认值为 **adKeyPrimary**。  
  
## <a name="remarks"></a>备注  
 对于已追加到集合的 [键](./key-object-adox.md) 对象，此属性是只读的。
