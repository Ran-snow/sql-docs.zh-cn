# <a name="convert-function-limitations"></a>CONVERT 函数限制
如果类型转换失败，则会将受影响的列设置为 NULL。  
  
 既不能将日期和时间戳数据类型转换为另一种数据类型 (也不能将其自身) 转换函数。
