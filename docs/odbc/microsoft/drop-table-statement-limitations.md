# <a name="drop-table-statement-limitations"></a>DROP TABLE 语句限制
当使用 Microsoft Excel 5.0、7.0 或97驱动程序时，DROP TABLE 语句将清除工作表，但不会删除工作表名称。 由于工作簿中仍存在工作表名称，因此不能用相同的名称创建其他工作表。
