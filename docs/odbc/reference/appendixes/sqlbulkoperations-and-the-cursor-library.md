# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations 和游标库
> [!IMPORTANT]  
>  此功能将在 Windows 的将来版本中删除。 避免在新的开发工作中使用此功能，并计划修改当前使用此功能的应用程序。 Microsoft 建议使用驱动程序的游标功能。  
  
 游标库不支持调用 **SQLBulkOperations**。 有关 **SQLBulkOperations** 的常规信息，请参阅 [SQLBulkOperations 函数](../../../odbc/reference/syntax/sqlbulkoperations-function.md)。
