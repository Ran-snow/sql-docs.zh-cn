# <a name="sqlsetparam-function"></a>SQLSetParam 函数
**度**  
 引入的版本： ODBC 1.0 标准符合性：已弃用  
  
 **摘要**  
 在 ODBC 2.0 中，ODBC 1.0 函数 **SQLSetParam** 已被 **SQLBindParameter** 取代。 有关详细信息，请参阅 [SQLBindParameter 函数](../../../odbc/reference/syntax/sqlbindparameter-function.md)。  
  
> [!NOTE]  
>  有关在 ODBC 2 时驱动程序管理器将此函数映射到的内容的详细信息。*x* 应用程序正在使用 ODBC 3。*x* 驱动程序，请参阅 [映射弃用的函数](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)。  
  
## <a name="remarks"></a>备注  
 如果你的应用程序将在64位操作系统上运行，请参阅 [ODBC 64 位信息](../../../odbc/reference/odbc-64-bit-information.md)。
