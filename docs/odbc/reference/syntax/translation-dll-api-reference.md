 本部分包含以下主题。  
  
-   [SQLDataSourceToDriver 函数](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)  
  
-   [SQLDriverToDataSource 函数](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)
