# <a name="cursors-event-category"></a>Cursors 事件类别
[!INCLUDE [SQL Server - ASDB](../../includes/applies-to-version/sql-asdb.md)]
   Cursors  事件类别包含用于监视游标行为的事件类。  
  
## <a name="in-this-section"></a>本节内容  
  
|主题|说明|  
|-----------|-----------------|  
|[CursorClose 事件类](../../relational-databases/event-classes/cursorclose-event-class.md)|说明应用程序编程接口 (API) 游标中发生的游标关闭事件。|  
|[CursorExecute 事件类](../../relational-databases/event-classes/cursorexecute-event-class.md)|说明 API 游标中发生的游标执行事件。|  
|[CursorImplicitConversion 事件类](../../relational-databases/event-classes/cursorimplicitconversion-event-class.md)|说明 API 或 [!INCLUDE[tsql](../../includes/tsql-md.md)] 游标中发生的游标隐式转换事件。|  
|[CursorOpen 事件类](../../relational-databases/event-classes/cursoropen-event-class.md)|说明 API 游标中发生的游标打开事件。|  
|[CursorPrepare 事件类](../../relational-databases/event-classes/cursorprepare-event-class.md)|说明 API 游标中发生的游标准备事件。|  
|[CursorRecompile 事件类](../../relational-databases/event-classes/cursorrecompile-event-class.md)|说明 API 游标中发生的游标重新编写事件。|  
|[CursorUnprepare 事件类](../../relational-databases/event-classes/cursorunprepare-event-class.md)|说明 API 游标中发生的游标未准备事件。|
